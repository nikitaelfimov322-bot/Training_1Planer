# Training_1Planer# Training Planner

## Автор
Никита

## Описание
GUI-приложение для планирования тренировок.
Позволяет:
- добавлять тренировки
- фильтровать по дате и типу
- сохранять данные в JSON

## Использование

1. Введите дату (YYYY-MM-DD)
2. Укажите тип тренировки
3. Введите длительность
4. Нажмите "Добавить тренировку"

### Фильтрация:
- Введите тип или дату
- Нажмите "Применить фильтр"

## Пример теста
Добавьте:
Дата: 2026-05-01  
Тип: Бег  
Длительность: 30

Проверьте фильтр по типу "Бег"
